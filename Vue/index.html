# 介绍

这个不是Vue的教程，学习Vue建议去看官网教程。这个只是最近过滤Vue知识点时的记录

# vue.js是什么

vue.js 是一套用于构建用户界面的渐进式框架。与其它大型框架不同的是，Vue 被设计为可以自底向上逐层应用。Vue 的核心库只关注视图层，不仅易于上手，还便于与第三方库或既有项目整合，与Vue-Router和Vuex搭配实现负责应用。

# 编译器和运行时

  • 完整版： 同时包含编译器和运行时的版本
  • 编译器：用来将模板字符串编译成为 JavaScript 渲染函数的代码
  • 运行时：用来创建 Vue 实例、渲染并处理虚拟 DOM 等的代码。基本上就是除去编译器的其它一切。

如果你需要在客户端编译模板 (比如传入一个字符串给 template 选项，或挂载到一个元素上并以其 DOM 内部的 HTML 作为模板)，就将需要加上编译器，即完整版。

当使用 vue-loader 或 vueify 的时候，*.vue 文件内部的模板会在构建时预编译成 JavaScript。你在最终打好的包里实际上是不需要编译器的，所以只用运行时版本即可。

# Vue vs React

# 相同点

  • 将注意力集中在核心库，路由、状态管理等功能交给其他库，渐进式框架
  • Virtual DOM
  • 组件化

# 不同点

  • 在 React 应用中，当某个组件的状态发生变化时，默认它会以该组件为根，重新渲染整个组件子树
  • JSX vs Templates
    • JSX可以使用完全的js功能，使用更灵活
    • Templates 更易于理解，现有HTML应用更易于迁移Vue
  • Vue单文件系统
